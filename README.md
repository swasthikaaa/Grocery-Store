## Product Categories

- **Vegetables** - Fresh tomatoes, carrots, broccoli, and more
- **Fruits** - Apples, bananas, oranges, strawberries
- **Dairy** - Milk, cheese, yogurt, butter
- **Beverages** - Orange juice, coffee, tea, water

## How to Use

1. **Browse Products** - Scroll through all available products or use category filters
2. **Add to Cart** - Click "Add to Cart" button on any product
3. **View Cart** - Click the shopping cart icon in the navbar
4. **Manage Cart** - Use +/- buttons to adjust quantities or remove items
5. **Checkout** - Click "Checkout" to complete your order

## Technologies Used

- **HTML5** - Semantic markup

## Features Breakdown

### Navigation
- Sticky navbar with logo and menu
- Hamburger menu for mobile devices
- Shopping cart counter badge

### Product Display
- Grid layout that adapts to screen size
- Filter buttons for easy browsing
- Product cards with emoji icons, names, prices
- Hover effects and animations

### Shopping Cart
- Modal popup for cart view
- Add/remove products
- Adjust quantities
- Real-time total calculation
- Empty state message

### Responsive Breakpoints
- Desktop: Full layout with all features
- Tablet (768px): Adjusted grid and font sizes
- Mobile (480px): Single column layout, simplified navigation
